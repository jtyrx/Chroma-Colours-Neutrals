# Chroma Colours Neutrals

This repository has been upgraded from the original CodePen export into a standalone
Next.js App Router project with Tailwind CSS v4.

Original URL: [https://codepen.io/jtyrx/pen/PwGxzqe](https://codepen.io/jtyrx/pen/PwGxzqe)

## Stack

- Next.js 16.2.3
- React 19.2.5
- Tailwind CSS 4.2.2
- TypeScript 6.0.2

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Related project

The **Neutral System Builder** (OKLCH neutral ramps, design tokens, export) lives in its own repository: [github.com/jtyrx/neutral-system](https://github.com/jtyrx/neutral-system). Live demo: [neutral-system.vercel.app](https://neutral-system.vercel.app).

## Project Structure

- `app/` — App Router pages, components, and global Tailwind v4 styles
- `src/` and `dist/` — original static export kept for reference
